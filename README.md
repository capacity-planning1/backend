# Визуальный планировщик ресурсов для командной разработки. 

REST API, написанное на фреймворке FastAPI, которое помогает тимлидам студенческих проектов видеть реальную загрузку команды и планировать спринты на основе актуальных данных об учебной занятости участников.



## 🚀 Запуск проекта 

``` bash
python -m venv .venv
```

``` bash
source .venv/Scripts/activate`
```

``` bash
pip install uv
```

``` bash
uv sync
```

``` bash
fastapi dev main.py
```
